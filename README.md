

###这是什么

habitrack是一个习惯跟踪系统，用于习惯的培养和跟踪；  
本项目部分是为配合Yii+Mongo+Mysql+Redis+Memcache+Bootstrap学习；

###技术选用

*	**采用PHP Yii框架**
*	**数据库采用MongoDb+Mysql**
*	**缓存采用Redis+Memcache**
*	**前端框架采用Bootstrap**
